TEK publish metrics

This graph reflects data from the key-server for this realm, grouped by UTC day.

Total # TEKs published

This line tracks the total number of TEKs published to the key-server. This is expected to be similar to the number of tokens claimed on the verification server.

Requests with revisions

This line tracks the total number of publish requests that contained at least one TEK revision

Request missing onset

This line tracks the total number of publish requests where no onset date was provided. These request are not included in the onset to upload distribution.

Publish requests (by device operating system)

This graph shows counts of the number of publish requests broken down by the request operating system. Hover over columns to see the percentage for each OS segment.

EN days active before upload

This graph shows a distribution of the ages of temporary exposure keys by how many days old each TEK is. Move the slider to see how the distribution has changed over time. Each bar shows the number of keys that are less than N days old and greater than the previous bar. Eg. "4" shows keys between 3 and 4 days old.

Onset-to-upload distribution

This graph reflects the distribution of the time between the TEK's symptom onset time and when the key was uploaded. Move the slider to see how the distribution has changed over time. Each bar shows the number of keys whose symptom onset-to-upload is less than N days old but greater than the previous bar. Eg. "4" shows symptom-onset between 3 and 4 days old.
